# RPG UI

基于 **Vite + React + TypeScript + Tailwind CSS** 构建的 RPG 界面。

## 初始化

```bash
npm install
```

## 开发

启动本地开发服务器（默认 http://localhost:5173，支持热更新）：

```bash
npm run dev
```

## 打包

构建生产产物到 `dist/`（先做类型检查再打包）：

```bash
npm run build
```

预览打包后的产物：

```bash
npm run preview
```

## 格式检查

类型检查：

```bash
npm run typecheck
```

代码规范检查（ESLint）：

```bash
npm run lint
```

代码格式检查 / 自动格式化（Prettier）：

```bash
npm run format:check
npm run format
```
